## What's New in v2.10.0

Focus of this release: **AsyncAPI Operation Groups** end-to-end, plus deployment and migration hardening.

### AsyncAPI Operation Groups
- Create and manage operation groups for AsyncAPI the same way as for REST
- Backend and UI support for grouping; processor/build-task-consumer export for groups
- AsyncAPI parser/toolchain updates aligned with group export

### Platform hardening
- nginx client and proxy timeouts increased for large portal workloads
- Failed-build retention extended for diagnostics
- Migration vacuum removed from cleanup stage so it no longer blocks operations
- MCP stream session memory growth prevented on the backend

### Fixes
- Incorrect channel list when exporting AsyncAPI for selected operations
- Compare REST prefix groups: change severity shown with text annotation instead of markers only
- Doc View for REST specs with templated paths that differ only by path parameter name
