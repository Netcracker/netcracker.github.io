## What's New in v2.11.0

This release productizes the **AI Assistant (Honey)** and continues AsyncAPI quality and MCP tooling.

### AI Assistant (Honey)
- Backend AI chat productization with configurable enablement
- Portal UI for conversational assistance over package/version context
- Business metrics for MCP tools and AI chat usage

### AsyncAPI & quality
- AsyncAPI Quality Check in the UI
- `experimental` apiKind treated like `no-bwc` for change classification
- Builder version checks support migration mode (relaxed validation while migrations run)

### Spec & tooling
- Numeric `exclusiveMinimum` / `exclusiveMaximum` (OAS 3.1 / JSON Schema) across unifier, diff, doc-viewer, apispec-view
- MCP tool for api-linter results
- Node.js 24 LTS for class-view, apispec-view, api-doc-viewer, and related CI images

### Fixes
- Linter overview issue counts limited to active rulesets
- UI container adapted for `readOnlyRootFilesystem=true`
- Combiner option matching improvements in api-diff
- Global search default workspace preselection
