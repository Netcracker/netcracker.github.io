## What's New in v2.8.0

This release marks a major milestone: **AsyncAPI 3.0 is now fully supported** across all layers of the platform.

### AsyncAPI Full Support
- AsyncAPI 3.0 document page with full rendering (Info, Operations, Channels, Messages)
- SEND/RECEIVE operation badges with channel addressing
- Kafka and AMQP bindings at operation, channel, and message level
- Global search for AsyncAPI operations in both UI and backend
- Agent-based discovery of AsyncAPI services in Kubernetes
- Normalization and API change classification for AsyncAPI

### GraphQL Enhancements
- Global search for GraphQL operations (UI + backend)
- Quality Gate for GraphQL enabled via Spectral linting

### Quality & Governance
- REST API Quality Gate: Spectral linting for REST APIs now enabled and configurable
- GraphQL Quality Gate: Spectral linting enabled for GraphQL

### Developer Experience
- TypeScript 7.0 upgrade for `api-diff`, `api-processor`, `api-unifier`, and related libraries
- Sharable/Non-sharable meta-marker: new spec-level flag controls document export behavior
- Previous version dependency validation added to backend build pipeline
