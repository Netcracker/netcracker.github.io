## What's New in v2.12.0

Major catalog expansion: **MCP** and **DDL** contracts land in the Portal, with DDL schema rendering and AsyncAPI diff visualization.

### MCP & DDL contracts
- UI support for MCP and DDL contract types (basic publish/browse scenarios)
- api-processor builders for MCP and DDL contracts
- DDL data model and `DdlTableViewer` for database schema documents
- Backend/document APIs for MCP and DDL contract storage

### AsyncAPI Diff View
- Render AsyncAPI 3.0 changes in Doc View (api-doc-viewer + Portal E2E)
- Data-driven JSON Schema Object (JSO) diffs with extensions support

### Docs & platform
- Mermaid diagrams in Markdown document view
- Optional custom CA certificate mounts for backend and linter TLS outbound trust
- Secure TLS configuration for HTTP clients (backend, agent, linter)

### Fixes
- GraphQL API Changes and Deprecated tabs enabled in production mode
- YAML parsing reliability (`js-yaml` → `yaml` across TS stack)
- Snapshot name validation; package kind validation on group publish
- Global search respects selected group; moved-package export redirect
