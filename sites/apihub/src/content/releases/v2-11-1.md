## What's New in v2.11.1

Patch release focused on **search performance**, **auth**, **diff accuracy**, and **AI Assistant** operability.

### Search & indexes
- Global search performance improvements
- Removal of obsolete search indexes unused by `/api/v4/search`
- Invalid suspicious builds for operation search text fixed

### Auth & AI
- Package API keys work on endpoints without `packageId` in the path
- Authorization added for previously under-protected endpoints and MCP tools
- Honey (AI Assistant) can be hidden when backend `ai.chat.enabled` is false

### Diff & documentation
- Combiner branches matched by direct `$ref` origin (fewer false breaking changes for `allOf` → `oneOf`)
- Doc Viewer shows titles for `oneOf` property options when present

### Platform
- Migration post-check result correctness
- Playwright 1.60.0 to unblock CI
- Build-task-consumer swagger/healthcheck cleanup
